Questions à choix multiples

1 - Quelle est la différence entre une balise de type bloc et une balise de type inline ?

  • Une balise de type bloc permet de créer des blocs de glace
  • Une balise de type bloc prend toute la largeur de l'écran contrairement à une balise de type inline qui ne prend que la largeur nécéssaire
  • Une balise de type bloc ne peut contenir que des id
