Questions à choix multiples

1 - Quelle est la différence entre une balise de type bloc et une balise de type inline ?

  1. Une balise de type bloc permet de créer des blocs de glace
  2. Une balise de type bloc prend toute la largeur de l'écran contrairement à une balise de type inline qui ne prend que la largeur nécéssaire
  3. Une balise de type bloc ne peut contenir que des id

2 - Que doit-on faire pour avoir une liste non ordonnée avec des puces carrées ?

  1. <ul type="square" >
  2. <ul form="carre" >
  3. <ol type="square" >

3 - Quel est l'attrbut de base de la balise video pour afficher les commandes de visualisation ?

  1. control
  2. controler
  3. controls

4 - Quelle est la bonne syntaxe pour ecrire un lien ?

  1. <a link = "google.com" >
  2. <a to = "google" >
  3. <a href = "google.com" >

5 - Que signifie HTML ?

  1. Hyperlinks and Text Markup Language
  2. Home Tool Markup Language
  3. Hyper Text Markup Language

6 - Quel est le titre de plus grande importance ?

  1. <head>
  2. <h1>
  3. <h6>

7 - Quelle est la balise qui permet de faire un retour chariot ?

  1. <rc>
  2. <lb>
  3. <br>

8 - Quel balise utilise-t-on pour créer une liste ordonnées ?

  1. <ul></ul>
  2. <lo></lo>
  3. <ol></ol>

9 - Quelle est la proposition correcte pour créer un champ de saisi

  1. <input type="field">
  2. <input type="textfield">
  3. <input type="text">

10 - Quelle balise HTML determine le titre d'une page?

  1. <head>
  2. <meta>
  3. <title>
